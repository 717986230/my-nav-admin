# uuoo.site 导航网站

一个现代化的导航网站，支持数据库存储、文件上传和微信小程序。

## 🌐 访问地址

- 网站: https://uuoo.site
- 管理后台: https://uuoo.site/admin
- API: https://api.uuoo.site

## ✨ 功能特点

### 后端
- ✅ SQLite 数据库
- ✅ 文件上传（图标）
- ✅ REST API
- ✅ 点击统计
- ✅ 分类管理

### 前端
- ✅ 现代化 UI
- ✅ 响应式设计
- ✅ 搜索功能

### 微信小程序
- ✅ 分类展示
- ✅ 搜索功能
- ✅ 分享功能

## 🚀 部署

详见 [DNS-CONFIG.md](./DNS-CONFIG.md)

## 📦 项目结构

```
my-nav-admin/
├── server.js          # Express 服务器
├── public/            # 前端文件
├── uploads/           # 上传的图标
├── vercel.json        # Vercel 配置
└── package.json       # 依赖
```
